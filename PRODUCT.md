# PRODUCT.md — M-International

## Что это

M-International — международная компания в сфере натурального здравоохранения (БАДы, нутрицевтики). Бизнес-модель: MLM (сетевой маркетинг) с гибридной системой вознаграждений.

## Продукт

**Основной продукт:** Веб-сайт-лендинг (m-international.kz) — единственная точка контакта с потенциальными клиентами и партнёрами.

**Целевая аудитория:**
- Люди 30-55 лет, заинтересованные в здоровье/долголетии (клиенты)
- Потенциальные партнёры для MLM-бизнеса (25-50 лет)
- География: Казахстан, СНГ, Монголия, Турция, 50+ стран

**Регистр:** Brand (маркетинговый лендинг). Дизайн И есть продукт.

## Голос бренда

- **Тон:** Уверенный, но не агрессивный. Научный, но доступный.
- **Слова-ассоциации:** Природа, наука, качество, глобальность, доверие
- **НЕ является:** Дешёвый MLM, агрессивные продажи, псевдонаука
- **Позиционирование:** Премиальные натуральные добавки с международными сертификатами

## Конкурентные преимущества

1. Международные сертификаты (GMP, ISO, Halal, FDA)
2. 9 типов бонусов в MLM-системе
3. Присутствие в 50+ странах
4. 15 лет на рынке
5. 100% натуральные ингредиенты

## Структура сайта

1. **Hero** — Главный экран с видео/изображением природы, заголовок, CTA
2. **Certificates** — Бегущая строка сертификатов (marquee)
3. **Products** — Каталог продукции (8 карточек)
4. **About** — О компании: миссия, видение, ценности, таймлайн
5. **Business** — MLM-предложение: преимущества, шаги
6. **CTA** — Финальный призыв к действию
7. **Footer** — Контакты, ссылки, рассылка

## Технические ограничения

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- next-intl v4 (ru, en, kk)
- Framer Motion + GSAP + Lenis
- Шрифты: Unbounded (заголовки), Onest (текст)
- Деплой: Vercel

## Дизайн-ограничения (СНГ-адаптация)

- Заголовки компактнее (clamp 2rem→3.5rem, НЕ 3.5rem→8rem)
- line-height ≥ 1.1 для заголовков (не 0.95)
- letter-spacing: normal (не negative) для кириллицы
- Без snap-scroll
- Без split-screen CTA
- Короткие тексты: 1-3 строки на экран
- Визуал/видео/интерактив вместо длинных текстовых блоков

## Ключевые метрики успеха

- Конверсия в заявку/регистрацию
- Время на сайте
- Скролл-глубина
- Bounce rate
